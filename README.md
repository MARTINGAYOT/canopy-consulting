# Canopy Consulting Website

**"Deep roots. Wide reach. Unstoppable growth."**

A static multi-page website for Canopy Consulting, a business consulting firm based in Belize.

---

## File Structure

```
canopy-consulting/
├── index.html              ← Homepage
├── css/
│   └── main.css            ← All shared styles + design tokens
├── js/
│   ├── main.js             ← Shared interactions (nav, scroll reveal)
│   └── components.js       ← Nav + footer injected into every page
├── pages/
│   ├── about.html          ← About / Our Story
│   ├── journey.html        ← The 5-Phase Growth Journey
│   ├── services.html       ← All packages + pricing
│   ├── case-study.html     ← Live case study documentation
│   └── contact.html        ← Contact form + FAQ
└── assets/                 ← Images, logo files (add yours here)
```

## Brand

| Token         | Value     | Usage                        |
|---------------|-----------|------------------------------|
| Forest Green  | `#1A4D3A` | Primary — logo, headings     |
| Dark Slate    | `#2C3E35` | Secondary — body text        |
| Amber Gold    | `#B8860B` | Accent — rules, badges, CTA  |
| Off-White     | `#F7F7F5` | Background                   |
| Mid Green     | `#3D7A5A` | Taglines, secondary text     |

**Fonts:** Playfair Display (headings) · Cormorant Garamond (lead text) · Lato (body/UI)

## Testing on GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://yourusername.github.io/canopy-consulting/`
